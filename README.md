<h1>Math 3250 - Intro to Data Analysis</h1>

This class requires R and RStudio to be installed. The following are instructions to install on fedora linux.

```
sudo dnf install R
```

```
cd ~/Downloads/
curl https://download1.rstudio.org/electron/rhel9/x86_64/rstudio-2026.01.0-392-x86_64.rpm --output rstudio.rpm
sudo dnf install ./rstudio
```
